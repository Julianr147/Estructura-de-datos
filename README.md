## Estructura-de-datos 📈📉

**¿Que es? ❓**
- En Ingeniería de Sistemas, una estructura de datos es una forma de organizar y almacenar información de manera eficiente para que pueda ser utilizada y manipulada de manera óptima en los programas. Permite gestionar datos de diferentes tipos (como enteros, cadenas o listas) y realizar operaciones como búsqueda, inserción, eliminación y ordenación, facilitando el desarrollo de soluciones informáticas eficaces y escalables.
